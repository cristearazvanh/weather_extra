# MagicMirror² Weather Module with OpenWeatherMap OneCall enhancements and extra data

Weather_extra module (left) vs Onecall module (right) with same design https://github.com/hangorazvan/onecall

<img style="height:1000px" src=https://github.com/hangorazvan/weather/blob/master/weather.png> <img style="height:1000px" src=https://github.com/hangorazvan/weather/blob/master/onecall.png>

#### Current weather (onecall)

- humidity
- sunrise
- sunset
- temperature
- units
- weatherType
- windDirection
- windSpeed
- feels like

plus
- dew point
- UV index
- mmHg pressure
- visibility
- description
- alerts

#### Weather forecast (hourly and daily onecall)

- date
- maxTemperature
- minTemperature
- rain
- units
- weatherType

plus
- feels like
- dew point
- UV index
- mmHg pressure
- visibility
- humidity
